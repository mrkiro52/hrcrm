# HR CRM - Система управления кандидатами

Современное веб-приложение для управления кандидатами в HR-отделе с интерфейсом в стиле Atlassian Jira.

## Технологический стек

- **React 18+** с TypeScript
- **Vite** - сборщик
- **Tailwind CSS** - стили
- **React Router v6** - маршрутизация
- **@tanstack/react-table** - таблицы с сортировкой и фильтрацией
- **react-beautiful-dnd** - Drag & Drop для канбан-доски

## Установка зависимостей

Выполните следующие команды в терминале:

```bash
# Установка всех зависимостей
npm install

# Или используйте yarn/pnpm
yarn install
# pnpm install
```

## Запуск приложения

```bash
# Режим разработки
npm run dev

# Сборка для production
npm run build

# Предпросмотр production сборки
npm run preview

# Линтинг кода
npm run lint
```

После запуска приложение будет доступно по адресу: http://localhost:3000

## Основные возможности

### 1. Канбан доска
- Визуализация кандидатов по статусам
- Drag & Drop для изменения статусов
- Карточки с аватарами и основной информацией

### 2. Список кандидатов
- Таблица с сортировкой по всем колонкам
- Фильтрация по позиции и статусу
- Поиск по имени
- Клик на строку открывает детальную информацию

### 3. Карточка кандидата
- Просмотр и редактирование данных
- Форма с валидацией
- История комментариев (демо)
- Удаление кандидата

### 4. Позиции
- Список доступных позиций
- Информация об отделах и статусах

## Моковые данные

Приложение использует статические данные из `src/services/mockData.ts`. Все действия (создание, редактирование, удаление) выводятся в консоль браузера, но не сохраняются между перезагрузками.

## Дальнейшая разработка

Для интеграции с реальным бэкендом:
1. Создайте API сервис для работы с сервером
2. Замените моковые данные на реальные запросы
3. Добавьте управление состоянием (Redux Toolkit/Zustand)
4. Реализуйте аутентификацию пользователей
